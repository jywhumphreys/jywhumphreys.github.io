BruinsOnBoard
BruinsOnBoard is a rideshare web application developed for UCLA students to share rides to and from LAX at a reduced cost. The app was built using the MERN stack (MongoDB, ExpressJS, ReactJS, and NodeJS).
GitHub Link
Key features:
  1. Ride arrangement: Users can create, find, and join rides using a user-friendly calendar interface. The react-big-calendar and react-datepicker libraries were used to create a clear and efficient user experience.
  2. User profiles: The app features an authenticated signup and login system. Users can create profiles linked to their email, add a bio, and search for other users dynamically.
  3. Email confirmations: When a user creates or joins a ride, the app sends a confirmation email with ride details. Group members also receive email updates when a new user joins or leaves the ride.
  4. Secure authentication: User passwords are encrypted using a JWT key, ensuring security. The authentication system verifies valid JWT tokens for API calls to fetch data from the database.
  5. User search functionality: Users can search for other registered members, making it easier to connect with potential rideshare partners.
